Core IT Job Roles

Microsoft365 Administrator

Learn to manage productivity and collaboration tools like Teams, Exchange Online, and SharePoint

MS Azure Administrator

Learn to manage virtual machines, cloud infrastructure, advanced identity and access management

Software Developer (Technical)

Learn to research, design, and develop applications by writing code, testing, and debugging

Data Analyst

Learn to gather, clean, and interpret data to help organizations make better business decisions

Cyber Security Specialist

Learn to protect organizations by assessing risks, designing security strategies, and building firewalls

Cloud Architect

Learn to build scalable cloud solutions, and ensure it meets business needs for performance, and cost-efficiency

AI ML Engineer

Learn to develop, and deploy AI and ML systems to solve complex problems by building models, and training intelligent algorithms

DevOps Engineer

Learn to ensure security, reliability, and performance of applications through monitoring and collaboration with other teams

Mobile App Developer

Learn to design and build sophisticated Native apps (iOS, Droid), Web apps and Hybrid apps

Network Administrator

Learn to securely and efficiently manage and maintain the organization's computer networks

UI/UX Designer

Learn to design digital products that are both visually appealing (UI) and easy to use (UX) for delightful user experiences

Technical Writer

Learn to create clear, and user-friendly documentation such as writing user manuals, how-to guides, and online help articles

Automation Engineer

Learn to implement automated processes and systems to increase efficiency of applications

Business Analyst

Learn to process, interpret and document business processes through analysis of data

Project Manager

Learn to plan, organize, and oversee large-scale projects to ensure successful outcomes
